# Exploratory Data Analysis (EDA)
The complete EDA guide + hands-on projects: data cleaning, univariate/bivariate/multivariate analysis, statistical validation, and visual storytelling. Python, SQL, Power BI. Building this while preparing for Data Analyst roles.

---

## The EDA Checklist I follow

1. **First Look** — shape, dtypes, understand what each column means
2. **Data Cleaning** — missing values, duplicates, wrong dtypes, outliers
3. **Univariate Analysis** — distributions, spread, value counts
4. **Bivariate / Multivariate Analysis** — correlations, groupby, crosstabs
5. **Statistical Validation** — back patterns with actual tests, not just charts
6. **Summarize & Communicate** — findings + a clear recommendation

More detail on each step will be added as I write it up properly.
